# My dotfiles
How to grab this repo:
WARNING: WILL PROBABLY OVERWRITE YOUR CONFIGS
```
git clone --bare <git-repo-url> $HOME/.dotfiles
source ~/.zshrc
config config --local status.showUntrackedFiles no
```
